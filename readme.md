##Meeting of the Minds
Meeting of the Minds is a full stack web application designed to facilitate the use of quick, easy-to-administer reading assessments with high utility for instructional decision making. It is designed to assist teachers in easily sorting their students into instructional reading groups and monitor as well as make changes to reading instruction based on the reading data.
